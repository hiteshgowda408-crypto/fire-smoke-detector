# Fire and Smoke Detector using Arduino

## Overview
This project detects fire and smoke using an MQ2 sensor and alerts using a buzzer.

## Components Used
- Arduino
- MQ2 Gas Sensor
- Buzzer

## Working
The MQ2 sensor detects smoke/gas levels. When the threshold is exceeded, the buzzer is activated.

## Applications
- Home safety systems  
- Industrial safety  
- Vehicle fire detection systems
